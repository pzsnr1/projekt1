----------------------------------------------------
Pobieranie projektu z github na komp. local
git clone nazwa_repozytorium
np.:
git clone https://github.com/pzsnr1/projekt1.git
-----------------------------------------------------

git status - prezentuje stan pliku w git
To polecenie powie nam na jakim branch(gałąź) jesteśmy i czy
mamy coś do commitowania

------------------------------------------------------

Dodawanie do repozytorium danych
git add . (dodaje do repo wszystko co zostało zmodyfikowane)
git add nazwa_pliku (dodaje tylko ten jeden plik)

--------------------------------------------------------

Wstawiamy message do commita
git commit -m "treść commita"

---------------------------------------------------------
git log - prezentuje wszystkie commity
